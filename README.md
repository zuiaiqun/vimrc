# vimrc
personal vimfile

## install
1. mkdir -p ~/.vim/bundle
2. install the plugin `Vundle`:   `git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`
3. exceute the command `Bundle Install`. Then, everything works.
## uses
* enter F5 to compile
* enter F6 to exceute
